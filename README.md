## Descripción

**MyGym2** es una aplicación pensada para ayudar en la gestión de un gimnasio. El objetivo es automatizar y facilitar tareas como el registro de socios, control de plantel, pagos y estadísticas del gimnasio.

> Este proyecto se desarrolla como parte de mi formación universitaria y está en constante evolución.

---

## Tecnologías utilizadas

- Lenguaje: C#
- Base de Datos: My SQL
